# v0.2.0

## ✨ 새 기능

### 블록 기반 에디터
- 한 메모 안에서 줄마다 텍스트 / 체크리스트 / 순서 리스트를 혼합 가능
- 헤더 서식 버튼으로 현재 커서가 있는 줄의 타입 변환
- Enter로 줄 분리, 줄 맨 앞 Backspace로 위 줄과 병합
- ↑↓ 방향키로 줄 간 이동

### 메모 카드 색상 변경
- 헤더 좌측 색상 도트 클릭으로 언제든 색상 변경 가능

### 카테고리 지정
- 메모 카드에서 개인 / 업무 / 아이디어 카테고리 직접 선택 가능

### Cmd+A 전체 선택
- Cmd+A 두 번으로 메모 전체 내용 선택 및 클립보드 자동 복사
- 복사 완료 시 토스트 알림 표시

## 🐛 버그 수정

### 줄바꿈 플리커 수정
- 빠른 입력 후 Enter 시 내용이 잠깐 되돌아가던 문제 수정 (낙관적 업데이트 적용)

### 메모 내용 카드 밖으로 넘치던 문제 수정
- 메모 카드에 `overflow-hidden` 추가
