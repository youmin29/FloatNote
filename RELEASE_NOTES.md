# v0.1.0

> 🎉 첫 번째 릴리즈

## ✨ 새 기능

### 파스텔 메모 카드
- 옐로우 · 핑크 · 라벤더 · 민트 · 블루 · 피치 6가지 파스텔 컬러 지원
- 새 메모 생성 시 색상 자동 순환
- 팝인 / 팝아웃 애니메이션

### 자유 배치
- 화면 어디든 드래그로 자유롭게 이동
- 우하단 핸들로 크기 조절
- 상단 드래그 영역으로 앱 창 이동
- 캔버스 더블클릭으로 해당 위치에 메모 생성

### 3가지 작성 모드
- **텍스트** — 자유 메모, 내용에 따라 자동 높이 확장
- **체크리스트** — 체크 시 부드러운 취소선 애니메이션
- **순서 리스트** — 드래그로 항목 순서 재배치

### 메모 관리
- 핀 고정 (항상 최상단) / 복제 / 아카이브 / 삭제
- 제목 편집 및 타임스탬프 표시

### 검색 & 필터
- 실시간 메모 내용 검색
- 컬러 필터 · 카테고리(개인 / 업무 / 아이디어) 필터

### 단축키
- `Cmd+Shift+N` — 새 메모 즉시 생성

### 로컬 데이터 영속성
- better-sqlite3 기반 로컬 DB — 앱 재시작 후에도 메모 유지
